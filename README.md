# The Bookworm's Burrow – Full Stack Online Book Library

The Bookworm's Burrow is a dynamic online book library web application developed to provide users with an engaging platform for browsing, reviewing, and exploring books across multiple genres. The project was designed with a warm user-friendly interface and integrated backend database functionality.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

## Core Features
- User registration and login authentication
- Dynamic book category filtering
- Book review and ratings section
- Responsive homepage with latest books
- Sidebar book details with author information
- SQL database integration
- Lazy image loading for performance optimization

## Project Purpose
This project was developed as a software engineering web development coursework to demonstrate full stack integration, relational database connectivity, and responsive front-end design.
